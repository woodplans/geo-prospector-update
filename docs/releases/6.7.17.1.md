# Geo Prospector 6.7.17.1

Version 6.7.17.1 improves scraping coverage and maximizes unique results after duplicate filtering.

## New data limits

Users can select:

- 5 data
- 10 data
- 20 data
- 50 data
- 100 data
- 200 data
- 300 data
- 400 data
- 500 data

## Maximum results

The selected number is a target. The final result depends on the number of relevant businesses available on Google Maps.

## Duplicate filtering

When Duplicate Filter is enabled, businesses collected previously are skipped. Geo Prospector continues checking additional candidates until the requested target is reached or available candidates are exhausted.

## Time estimates

Time estimates adjust automatically according to the selected data count and Email Finder status. Actual processing time depends on device performance, connection speed, Google Maps response, website availability, and duplicate density.
